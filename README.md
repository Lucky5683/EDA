# Exploratory Data Analysis (EDA) & Business Intelligence Project

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) and Business Intelligence analysis on coffee shop transaction datasets. The objective of the project was to uncover patterns, trends, and relationships within the data using statistical analysis, SQL querying, visualization techniques, and dashboard-oriented reporting.

The project demonstrates a complete analytics workflow from raw data exploration to business insight generation.

---

# Project Objectives

- Perform descriptive statistics and univariate analysis
- Explore customer and transaction behavior patterns
- Answer business questions using SQL queries
- Analyze relationships between variables using visualizations
- Create dashboard-oriented business insights

---

# Descriptive Statistics & Univariate Analysis

The following analyses were performed:

- Statistical summaries of numerical data
- Product frequency analysis
- Payment method distribution analysis
- Revenue trend analysis
- Histogram and bar chart visualizations

### Visualizations Created
- Histograms
- Bar Charts
- Scatter Plots
- Correlation Heatmaps
- Revenue Trend Charts

---

# SQL Business Questions

SQL queries were written to answer important business questions such as:

1. Top 5 coffee products by revenue
2. Total revenue by payment method
3. Average transaction amount
4. Peak sales hour
5. Product-wise transaction count
6. Monthly revenue trends
7. Revenue by day of week
